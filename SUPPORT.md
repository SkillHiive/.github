# Support

Thanks for using SkillHiive.

Before opening an issue, please make sure you've checked the documentation and searched existing discussions.

## I found a bug

Please open a GitHub Issue with:

- A clear description
- Steps to reproduce
- Expected behaviour
- Screenshots or logs (if applicable)
- Your environment

The more information you provide, the easier it is for us to investigate.

---

## I have a feature idea

Feature requests are welcome.

When suggesting new ideas, please explain:

- The problem you're trying to solve
- Why existing functionality isn't sufficient
- How the proposal aligns with SkillHiive's philosophy

We prioritize features that encourage healthier communities, collaboration, and intentional interaction.

---

## I need help

If something isn't working as expected:

- Check the documentation
- Search existing Issues
- Search Discussions

If you still can't find an answer, start a new Discussion.

---

## Security Issues

Please **do not** report security vulnerabilities publicly.

See our [Security Policy](SECURITY.md).

---

## Our Commitment

SkillHiive is an open-source project maintained by people donating their time and energy.

We'll always do our best to help, but please remember that responses may take time.

We appreciate your patience and your support.
